Wordpress plugin for managing Open contracting data standard records. Plugin made for Ojo con mi pisto.

== Description ==

This plugin is intended to be used in Ojo con mi pisto (OCMP) website about public works monitoring. This is not intended for government use, yet. It could, however be fitted to such purpose in next versions. For now it only suits the needs of OCMP's website.

== Installation ==

1. Upload `wp-ocds.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
